# ethereum-mining-calculator
Ethereum Mining Profitability Calculator

## Features
### Basic
#### Network stats
Network hashrate & blocktime are calculated over the last 64 blocks.

### Price calculation
[ETH=USD](http://coinmarketcap.com/currencies/ethereum/)
